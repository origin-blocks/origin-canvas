# Getting Started

Use this guide to install Origin, activate the theme, and make your first edit in the WordPress Site Editor.

## Install Origin

In your WordPress dashboard, go to Appearance > Themes > Add New > Upload Theme.

Choose the Origin theme ZIP file, then select Install Now.

## Activate the Theme

After WordPress finishes installing the theme, select Activate.

Origin becomes the active theme for your site and its templates, patterns, styles, and theme settings become available in the editor.

## Open the Site Editor

Go to Appearance > Editor.

The Site Editor is where you can edit templates, template parts, styles, and site-wide design settings for a block theme.

## Choose a Starting Point

Start with the included templates and patterns. Templates control page structure, while patterns give you editable sections and layouts you can insert into pages or template areas.

After inserting a pattern, replace the sample content with your own text, links, and images.
