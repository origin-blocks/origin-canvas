# Patterns and Templates

Origin includes patterns and templates so you can start from designed sections and site structures instead of building every layout from scratch.

## Patterns

Patterns are reusable page sections and layouts that can be inserted into pages, posts, templates, or template parts.

After inserting a pattern, you can edit its text, images, links, layout, and block settings.

## Templates

Templates control page structure by content type.

For example, a page template controls the layout for pages, while archive and search templates control listing views.

## Template Parts

Template parts are reusable sections shared across templates.

Headers, footers, and sidebars are common template parts. Editing one can update every template that uses it.

## Editing Safely

Before major experiments, duplicate or save a copy of patterns, templates, or template parts when appropriate.

This makes it easier to test changes while keeping a known starting point available.
