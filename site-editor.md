# Working in the Site Editor

Origin Canvas is a block theme, so most site-wide editing happens in the WordPress Site Editor.

## Styles

Use Styles to choose or adjust site-wide colors, typography, spacing, and other design settings.

Style changes affect the whole site unless a block, template, or page has more specific settings applied.

## Templates

Templates control page types such as pages, posts, archives, search results, and 404 pages.

When you edit a template, you are changing the structure used by every page or post that relies on that template.

## Template Parts

Template parts are reusable site sections such as the header, footer, and sidebar.

Editing a template part can affect every template where that part appears.

## Saving Changes

The Site Editor saves template and style changes separately from normal post or page content.

Before saving, WordPress may show which templates, template parts, or styles are about to be updated. Review that list when making site-wide changes.

## If Changes Do Not Appear

If a change does not appear on the front end, clear your browser cache and refresh the page.

Previously saved Site Editor customizations can also override theme defaults. Check whether the template, template part, or style has already been customized in the editor.
