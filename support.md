# Support

Use GitHub Issues for Origin Theme support.

## Get Help

Open an issue at [Origin Theme GitHub Issues](https://github.com/origin-blocks/origin-theme/issues).

## What to Include

Include the WordPress version, Origin version, browser, and where the issue appears.

Screenshots are helpful when the issue is visual. Steps to reproduce are helpful when the issue depends on a specific editing flow.

## Before Opening an Issue

When practical, test with only Origin active and no extra plugins enabled.

Also check whether the issue comes from custom Site Editor changes, a plugin, or content added to a specific page or template.
