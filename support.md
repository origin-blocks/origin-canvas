# Support

Use GitHub Issues for Origin Canvas support.

## Get Help

Open an issue at [Origin Canvas GitHub Issues](https://github.com/origin-blocks/origin-canvas/issues).

## What to Include

Include the WordPress version, Origin Canvas version, browser, and where the issue appears.

Screenshots are helpful when the issue is visual. Steps to reproduce are helpful when the issue depends on a specific editing flow.

## Before Opening an Issue

When practical, test with only Origin Canvas active and no extra plugins enabled.

Also check whether the issue comes from custom Site Editor changes, a plugin, or content added to a specific page or template.
